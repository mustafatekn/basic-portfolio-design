Yaptığım İşler

LinkedIn Klon

Firebase veritabanı kullandığım LinkedIn klonu projemin backendini NodeJS ile yazdım. Frontend için React Redux'ın yazıldığı dönem için güncel ancak günümüz için ilkel kalan bir versiyonunu kullandım. Elementleri elimden geldiğince html ve bootstrap kütüphanesinden yararlanarak oluşturmaya çalıştım, özellikle önerilen kullanıcılar kısmında css kullandım. Aslında tasarımı ilk başta html css bilgimi geliştirmek için hiç bootstrap kullanmadan yapmıştım, sonra da bootstrap bilgimi artırmak için bootstrap kullanmıştım. Ancak profil sayfasındaki önerilen kullanıcılar kısmını bootstrap kullanarak da yapabilirdim ama css yazmışım ve neden böyle yaptığım hakkında hiçbir fikrim yok :)

Portfolyo

GraphQL api designını kullandığım portfolyo projemin backendini NodeJS, frontendini React ile yazdım. Elimden geldiğince verimli kullanmaya çalıştığım bootstrap kütüphanesinin yanı sıra görece fazla css kodu içeriyor. Tasarımıyla ön plana çıkan portfolyo projesinin aynı zamanda bir admin paneli bulunuyor ve panele giriş yapmayan kullanıcılar herhangi bir değişiklik yapamıyor.

E-Ticaret

Tamamen .NET Core öğrenmek için Udemy'den Sadık Turan - Komple Uygulamalı Web Geliştirme Eğitimi içerisinde gerçekleştirilen bir e-ticaret projesidir. "Şifremi Unuttum" maillerinden kredi kartı entegrasyonuna kadar birçok detaya değinilen bu projeyi kursu takip ederken her şeyi hocayla birlikte uygulayarak sonlandırdım. Dotnete yönelik çabalarımın olduğunu göstermek için github'a attığım bu projeye burada da yer vermek istedim.
